# 💬 PROMPTS.md — Best Prompts for Smile Stories AI

Use these prompts to get the most out of your AI dental content agent.

---

## 🎬 Video Ideas

```
Give me 10 YouTube video ideas about [topic] for a dental channel
```
```
What are trending topics in dentistry I could make videos about?
```
```
Give me video ideas that would reduce fear of going to the dentist
```
```
Suggest 5 "day in the life" video ideas for a dental practice
```

---

## 📝 Full Scripts

```
Write a full YouTube script about [topic], keep it under 8 minutes, warm and friendly tone
```
```
Write a script explaining [procedure] to someone who is nervous about it
```
```
Write a beginner-friendly script about why flossing matters
```
```
Write a script debunking common myths about [topic]
```

---

## 🖼️ Titles & Thumbnails

```
Give me 5 catchy YouTube titles for a video about [topic]
```
```
What thumbnail concept would work best for a video about [topic]?
```
```
Write a YouTube title that's curiosity-driven for a video about braces
```

```
Write an Instagram caption for a video about [topic]
```
```
Give me 3 tweet ideas to promote my new video about [topic]
```

---

## 🗓️ Content Planning

```
Create a 4-week YouTube content calendar for my dental channel
```
```
Plan a series of videos about the patient journey from first visit to treatment
```
```
What seasonal dental topics should I cover in [month]?
```

---

## 🧠 Tips for Best Results

- Be specific: mention the target audience (e.g. "for anxious patients", "for parents of kids")
- Mention length: "keep it under 5 minutes" or "make it a short-form reel"
- Mention tone: "funny and light", "serious and educational", "story-driven"
- Ask for variations: "give me 3 different versions of this title"
