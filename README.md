# 😁 Smile Stories AI V2

An AI agent deployed on [Blink Claw](https://blink.new) that generates engaging YouTube video ideas and scripts for a dental channel — educational, entertaining, warm, and reassuring.

## 🎯 What This Agent Does

- Generates YouTube video ideas tailored for a dental audience
- Writes full scripts with a warm, friendly tone
- Focuses on education + entertainment (edutainment)
- Helps demystify dental procedures to reduce patient anxiety

## 📁 File Structure

| File | Purpose |
|------|---------|
| `IDENTITY.md` | Agent mission, model, and filesystem layout |
| `SOUL.md` | Personality, tone, and behavior guidelines |
| `AGENTS.md` | Blink CLI reference and available skills |
| `TOOLS.md` | Environment-specific notes and setup |
| `HEARTBEAT.md` | Periodic task configuration |
| `USER.md` | Notes about the human using this agent |
| `PROMPTS.md` | Example prompts to get the best results |
| `VIDEO_IDEAS.md` | Saved video ideas and concepts |

## 🚀 Getting Started

1. Create a free account at [blink.new](https://blink.new)
2. Deploy this agent by uploading these files
3. Start chatting and generating dental content!

## 💡 Example Prompts

- *"Give me 10 YouTube video ideas about teeth whitening"*
- *"Write a script for a video explaining why root canals aren't scary"*
- *"Create an engaging title and thumbnail idea for a video about braces"*
